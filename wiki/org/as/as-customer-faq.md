# Keychron AS 고객 응대 FAQ

고객 문의 빈도순 10개 FAQ와 단계별 트러블슈팅 가이드.

> A/S 접수: <https://keychron.kr/as/>

---

## 주요 단축키 참조표

| 기능 | 단축키 |
|------|--------|
| RGB ON/OFF | `fn` + `Light` |
| RGB 효과 변경 | `fn` + `Light` 길게 누르기 |
| RGB 밝기 증가 | `fn` + `→` |
| RGB 밝기 감소 | `fn` + `←` |
| RGB 속도 증가 | `fn` + `↑` |
| RGB 속도 감소 | `fn` + `↓` |
| 공장 초기화 | `fn` + `J` + `Z` (4초 이상) |
| BT 페어링 모드 | `fn` + `1/2/3` (4초 이상) |
| 배터리 잔량 확인 | `fn` + `B` |
| Mac/Win 모드 전환 | 측면 스위치 |

## FAQ 1. 블루투스 끊김 해결

### 1단계: 기본 리셋
1. 키보드를 USB 케이블로 연결한 상태에서 `fn` + `J` + `Z` 4초 이상 (공장 초기화)
2. PC/Mac의 블루투스 설정에서 기존 "Keychron" 페어링 기록 삭제
3. `fn` + `1` (4초 이상) → 블루투스 LED 빠르게 깜빡이면 페어링 모드
4. PC/Mac에서 새로 검색하여 연결

### 2단계: OS 블루투스 초기화
- **Mac**: 시스템 설정 → Bluetooth → 고급 → "디버그" → 모든 기기 제거 후 재부팅
- **Windows**: 장치 관리자 → Bluetooth 어댑터 드라이버 제거 → 재부팅

### 3단계: 수신 환경 점검
- USB 3.0 허브/외장 하드 등 2.4GHz 간섭 장치 제거
- 키보드와 PC 간 거리 1m 이내 확인
- 다른 BT 기기 동시 연결 수 최소화

### MacOS Sequoia 특별 케이스
macOS Sequoia(15.x) 업데이트 이후 BT 끊김이 급증하는 경우:
1. 시스템 설정 → 개인정보 보호 및 보안 → Bluetooth → Keychron 앱 허용 확인
2. `/Library/Preferences/com.apple.Bluetooth.plist` 삭제 후 재부팅
3. NVRAM 초기화: 전원 종료 → 전원 버튼 + `Option` + `Command` + `P` + `R` (20초)
4. 증상 지속 시 Apple 지원 또는 A/S 접수 안내

## FAQ 2. 채터링(이중 입력) 해결

> 채터링: 키를 한 번 눌렀는데 두 번 이상 입력되는 현상

### 무선(Bluetooth/2.4GHz) 채터링
1. 공장 초기화 (`fn` + `J` + `Z`)
2. USB 유선으로 연결 후 증상 재현 테스트
3. 유선에서 정상이면 무선 신호 간섭 문제 → 수신 환경 점검

### 유선 채터링
1. 다른 USB 포트/케이블로 테스트
2. 다른 PC에서 테스트
3. 증상 지속 시 A/S 접수

### 핫스왑 모델 — 핀 접점 조정
1. 키캡 제거 → 스위치 풀러로 해당 스위치 분리
2. 스위치 핀(2개)이 휘었는지 확인 → 핀셋으로 똑바로 교정
3. 다른 정상 스위치와 교체 장착하여 테스트
4. 소켓 자체 불량 시 A/S 접수

---

## FAQ 3. 특정 키 입력 불량

### 핫스왑 모델 (3단계)
1. **스위치 교체**: 해당 키 스위치를 정상 키와 교환 → 스위치 문제인지 소켓 문제인지 분리
2. **핀 점검**: 스위치 핀 휨, 꺾임 확인 → 교정 후 재장착
3. **소켓 점검**: 여러 스위치에서 모두 불량이면 PCB 소켓 불량 → A/S 접수

### 솔더링 모델
- 사용자 직접 수리 불가
- 증상 확인 후 바로 A/S 접수 안내

---

## FAQ 4. 충전 불량 / 배터리 빠른 소모

### 충전 트러블슈팅 (4단계)
1. **케이블 교체**: 정품 또는 데이터 통신 가능한 USB-C 케이블 사용 (충전 전용 케이블 불가)
2. **충전원 변경**: PC USB 포트 대신 5V/1A 이상 어댑터 직접 연결
3. **충전 표시 확인**: 충전 중 LED (빨간색) → 완충 시 LED 꺼짐 또는 녹색
4. **완전 방전 복구**: 30분 이상 연결 후 전원 ON → LED 반응 확인

### 배터리 절약 팁
- 미사용 시 전원 스위치 OFF
- RGB 밝기 최소 또는 OFF (`fn` + `Light`)
- BT 자동 슬립 모드 활용 (5분 미사용 시 슬립)
- `fn` + `B`로 잔량 확인 습관화

## FAQ 5. Mac 환경 설정

### fn 키 동작 변경
- 시스템 설정 → 키보드 → "F1, F2 등의 키를 표준 기능 키로 사용" 체크

### 한/영 전환
- **K 시리즈**: `Caps Lock` = 한/영 (기본)
- **Q/V 시리즈**: 우측 `Command` 옆 `Globe` 키 사용
- VIA에서 키 매핑 변경 가능

### 화면 밝기/음량 조절
- `fn` + `F1`/`F2` = 밝기 조절
- `fn` + `F10`/`F11`/`F12` = 음소거/음량 조절
- Mac 모드(측면 스위치)로 전환 필수

### ~ (틸드) 키 입력
- Mac 레이아웃에서 `ESC` 아래 키가 `` ` `` / `~`
- 입력 소스가 "ABC"일 때만 정상 동작, "한국어" 입력 중에는 `₩` 출력
- 해결: `~/Library/KeyBindings/DefaultkeyBinding.dict` 파일에 `"₩" = ("insertText:", "\`");` 추가

## FAQ 6. LED/RGB 백라이트 불량

### 1단계: 소프트 리셋
1. `fn` + `Light` (ON/OFF 토글)
2. 공장 초기화 (`fn` + `J` + `Z`)

### 2단계: 개별 LED 확인
- 특정 키만 색상 다르거나 꺼져 있는 경우 → LED 칩 불량
- RGB 효과 변경(`fn` + `Light` 길게)으로 전체 점등 확인

### 3단계: 하드웨어 불량 판단
- 전체 LED 미점등 + 키 입력은 정상 → PCB LED 회로 불량 → A/S 접수
- 부분 LED 미점등 → 해당 LED 불량 → A/S 접수

---

## FAQ 7. 펌웨어 업데이트 / VIA 키매핑

### 1단계: 펌웨어 업데이트
1. <https://keychron.kr> → 지원(Support) → 펌웨어 다운로드
2. USB 유선 연결 필수 (BT로는 업데이트 불가)
3. 안내 문서의 순서대로 진행 (모델별 상이)

### 2단계: VIA 키매핑
1. <https://usevia.app> 접속 (Chrome/Edge 권장)
2. USB 유선 연결 → "Authorize device" 클릭 → 키보드 선택
3. 키 클릭 후 원하는 기능 매핑
4. VIA 미지원 모델은 Keychron Launcher 사용: <https://launcher.keychron.com>

### 3단계: VIA 인식 불가 시
- 다른 USB 포트/케이블 시도
- Chrome 브라우저 `chrome://device-log` 에서 HID 인식 확인
- 펌웨어를 최신으로 업데이트 후 재시도

## FAQ 8. USB 유선 연결 불량

### 1단계: 물리 점검
- 케이블 양쪽 끝 단자 확인 (먼지, 꺾임)
- 다른 USB-C 케이블로 교체 테스트

### 2단계: PC 측 점검
- 다른 USB 포트(후면 포트 권장) 시도
- USB 허브 제거, PC 직접 연결
- 장치 관리자에서 "USB 입력 장치" 인식 여부 확인

### 3단계: 키보드 측 점검
- 다른 PC에서 테스트
- 공장 초기화 후 재연결
- 증상 지속 시 USB-C 포트 불량 가능성 → A/S 접수

## FAQ 9. 소음 판단 기준

기계식 키보드는 구조적으로 타건 소음이 발생하며, 아래 기준으로 정상/불량을 판단합니다.

| 구분 | 설명 |
|------|------|
| **정상 소음** | 키 입력 시 균일한 클릭/탁 소리, 스태빌라이저 딸깍 소리 |
| **비정상 소음** | 특정 키만 과도한 금속 마찰음, 스프링 핑 소리, 비균일 소음 |
| **스태빌라이저 소음** | 큰 키(스페이스바, Shift, Enter) 특유의 딸깍/삐걱 → 윤활로 개선 가능 |

- 스위치 타입별 소음 차이: 적축(조용) < 갈축(중간) < 청축(클릭)
- **A/S 대상**: 하나의 키에서만 비정상적 소음 발생, 사용 중 갑자기 소음 변화
- **비대상**: 스위치 특성에 의한 일반 타건음, 스태빌라이저 소음 (사용자 윤활 권장)

## FAQ 10. 2.4GHz 동글 연결 불량

### 1단계: 재페어링
1. 동글을 PC USB 포트에 삽입 (USB 허브 사용 금지)
2. 키보드 모드를 2.4GHz로 전환 (측면 스위치)
3. 공장 초기화 (`fn` + `J` + `Z`)
4. 동글 리시버 리셋: 동글 뒷면 핀홀 버튼 3초 누르기 (해당 모델만)
5. 키보드 `fn` + `4` (4초 이상) → 동글 LED 깜빡이면 연결 완료

### 2단계: 환경 점검
- USB 3.0 포트 옆에 동글 삽입 시 간섭 → USB 2.0 포트 또는 연장 케이블 사용
- 동글과 키보드 거리 2m 이내 유지
- 증상 지속 시 동글 불량 가능성 → 동글 교체 또는 A/S 접수
