# 1. KEYCHRON

> 누적 접수 29,007건 기준 분석

## 1.1 수리 표준 프로세스 (6단계)

| 단계 | 내용 | 소요 시간 |
|------|------|-----------|
| 1. 접수 | 온라인 폼 또는 전화 접수, 증상 기록 | 즉시 |
| 2. 입고 | 제품 수령, 외관 검수, 시리얼 확인 | 당일 |
| 3. 진단 | 증상 재현 테스트, 원인 분류 | 1영업일 |
| 4. 수리/교체 | 부품 수리 또는 교체 진행 | 1~3영업일 |
| 5. 검수 | 전 키 입력 테스트, BT/유선 연결 확인 | 당일 |
| 6. 출고 | 고객 발송, 택배 번호 안내 | 당일 |

## 1.2 유상 수리 비용표

| 항목 | 비용 (VAT 포함) |
|------|-----------------|
| 공임비 (기본) | 20,000원 |
| 메인보드 (PCB) 교체 | 50,000원 |
| 배터리 교체 | 30,000원 |
| USB-C 포트 수리 | 25,000원 |
| 스위치 교체 (개당, 솔더링) | 3,000원 |
| 스태빌라이저 교체 | 5,000원 |
| 케이스/프레임 교체 | 모델별 상이 |
| 키캡 세트 교체 | 모델별 상이 |

> 보증기간 내 제조 결함은 무상 수리. 침수/외력/사용자 분해는 유상.

## 1.3 증상별 진단 가이드 (7개)

### (1) 블루투스 불량 — 4,300건
- **테스트**: 공장 초기화 → 재페어링 → 3개 슬롯 전부 테스트
- **판정**: 초기화 후에도 끊김/미연결 → BT 모듈 불량 → PCB 교체
- **주의**: MacOS Sequoia 호환 이슈 구분 (OS 문제는 수리 불가)

### (2) 키 입력 / 채터링 — 1위 증상
- **핫스왑**: 스위치 교체 → 소켓 접점 확인 → 소켓 납땜 수리
- **솔더링**: 키 스위치 탈납 → 재납땜 → PCB 패턴 점검
- **채터링 테스트**: <https://www.keyboardtester.com> 에서 50회 반복 입력

### (3) 배터리 / 충전
- **테스트**: 멀티미터로 배터리 전압 측정 (정상: 3.7V~4.2V)
- **판정**: 3.2V 이하 → 배터리 교체, 충전 IC 고장 시 PCB 교체
- **주의**: 배터리 팽창 제품은 즉시 격리 보관

### (4) LED 불량
- **테스트**: 공장 초기화 → 전체 LED 점등 모드 확인
- **판정**: 개별 LED 미점등 → LED 칩 교체 (솔더링), 전체 미점등 → LED 드라이버 IC 불량

### (5) 소음
- **정상 범위**: 스위치 타입별 기본 타건음, 스태빌라이저 소리
- **비정상**: 스프링 핑, 금속 마찰, 특정 키만 과도한 소음
- **수리**: 스위치 윤활, 스태빌라이저 교체/윤활, 폼 추가

### (6) 이물질/침수
- **침수**: 즉시 전원 OFF → 배터리 분리 → 48시간 자연 건조 → 부식 확인
- **이물질**: 분해 청소 → 접점 세정 → 재조립
- **주의**: 침수 이력 제품은 유상 처리 기본

### (7) 펌웨어
- **증상**: 특정 키 매핑 오류, VIA 미인식, 업데이트 실패
- **수리**: 최신 펌웨어 재설치 → ISP 플래싱 (긴급 시)
- **주의**: 사용자 VIA 설정 오류와 펌웨어 버그 구분

## 1.4 새제품 교환 기준

| 조건 | 처리 |
|------|------|
| 수령 후 7일 이내 + 제조 결함 | 새제품 교환 |
| 수령 후 7일 이내 + 사용자 과실 | 유상 수리 또는 반품 |
| 동일 증상 3회 이상 반복 수리 | 새제품 교환 검토 (팀장 승인) |
| 수리 불가 판정 (부품 단종 등) | 동급 모델 교환 또는 환불 |

## 1.5 핫스왑 vs 솔더링 체크리스트

| 점검 항목 | 핫스왑 | 솔더링 |
|-----------|--------|--------|
| 스위치 교체 | O (풀러 사용) | X (납땜 필요) |
| 소켓 점검 | 핀 접점, 소켓 고정 확인 | 납땜 상태 확인 |
| PCB 점검 | 소켓 파손 여부 | 패턴 손상 여부 |
| 스위치 핀 | 휨/꺾임 교정 | 해당 없음 |
| 수리 난이도 | 낮음 | 높음 (납땜 숙련 필요) |

## 1.6 출고 체크리스트

- [ ] 전 키 입력 테스트 (키보드 테스터)
- [ ] BT 3슬롯 연결 테스트
- [ ] USB 유선 연결 테스트
- [ ] 2.4GHz 동글 연결 테스트 (해당 모델)
- [ ] LED 전체 점등 확인
- [ ] 충전 상태 확인 (50% 이상 충전 출고)
- [ ] 외관 청소 및 스크래치 확인
- [ ] 부속품 동봉 (케이블, 키캡풀러, 스위치풀러)
- [ ] 수리 내역서 동봉

# 2. AIPER

> 누적 접수 2,182건 기준 분석

## 2.1 처리 프로세스 (10단계)

| 단계 | 내용 |
|------|------|
| 1. 접수 | 전화/메일 접수, 증상 및 구매 정보 확인 |
| 2. 우선순위 분류 | 즉시/당일/일반 분류 |
| 3. 증상 기록 | CRM에 증상 코드 입력 |
| 4. 유무상 판단 | 보증기간/과실여부 확인 |
| 5. 입고 안내 | 발송 안내 (택배 착불/선불 구분) |
| 6. 입고/검수 | 외관 검수, 시리얼 확인 |
| 7. 진단/수리 | 증상 재현 → 수리 또는 부품 교체 |
| 8. 검수 | 작동 테스트 (풀 주행 시뮬레이션) |
| 9. 출고 | 고객 발송 + 택배 번호 안내 |
| 10. 사후 관리 | 3일 후 사용 만족도 확인 전화 |

## 2.2 우선순위 분류

| 등급 | 기준 | 대응 |
|------|------|------|
| **즉시** | 안전 관련 (배터리 팽창, 과열, 누수) | 즉시 회수, 당일 보고 |
| **당일** | 수령 7일 이내 불량, VIP 고객 | 당일 내 입고 안내 |
| **일반** | 일반 수리 접수 | 순서대로 처리 |

## 2.3 증상별 진단 (7개 Case)

### Case 1. 충전 불량 — 43.4% (947건)
- **점검**: 충전기/케이블 교체 → 충전 포트 이물질 → 배터리 전압 측정
- **판정**: 충전 IC 고장 → 메인보드 교체, 배터리 열화 → 배터리 교체
- **주의**: 방수 등급 제품 — 충전 포트 건조 상태 필수 확인

### Case 2. 전원/배터리 — 13.7% (299건)
- **점검**: 전원 버튼 동작 → 배터리 전압 → 메인보드 전원 회로
- **판정**: 배터리 3.0V 이하 → 교체, 전원 IC 불량 → 메인보드 교체

### Case 3. 부품 파손 — 11.2% (244건)
- **점검**: 외관 파손 부위 확인, 파손 원인 분석
- **판정**: 브러시/필터/바퀴 → 부품 교체, 본체 파손 → 유상 수리 또는 교환

### Case 4. 주행 이상
- **점검**: 바퀴 이물질 → 센서 청소 → 주행 모터 테스트
- **판정**: 센서 불량 → 센서 교체, 모터 불량 → 모터 교체

### Case 5. 전면 작동 불량
- **점검**: 전원 ON → 모든 기능 미작동 확인
- **판정**: 메인보드 불량 → 교체, 방수 씰 침투 → 부식 확인 후 수리

### Case 6. 흡입력 저하
- **점검**: 필터 막힘 → 펌프/모터 소리 → 흡입 경로 이물질
- **판정**: 필터 교체 → 펌프 교체 → 메인보드 확인

### Case 7. 소음
- **점검**: 정상 작동 소음 vs 비정상 소음 구분
- **판정**: 모터 베어링 마모 → 모터 교체, 이물질 → 제거

## 2.4 교환 처리 기준

| 조건 | 처리 |
|------|------|
| 수령 7일 이내 + 제조 결함 | 새제품 교환 |
| 동일 증상 2회 반복 | 새제품 교환 검토 |
| 수리 불가 (부품 미확보) | 동급 교환 또는 환불 |
| 안전 관련 결함 | 즉시 회수 + 교환 |

# 3. GTGEAR

> 취급 브랜드: MOZA, Fanatec, Logitech G, Playseat

## 3.1 취급 가능/불가 브랜드

| 구분 | 브랜드 |
|------|--------|
| **취급 가능** | MOZA, Fanatec, Logitech G, Playseat |
| **취급 불가** | Thrustmaster, Simucube, Heusinkveld 등 미입점 브랜드 |

> 미입점 브랜드 문의 시 해당 브랜드 공식 AS로 안내

## 3.2 수리 표준 프로세스 (6단계)

| 단계 | 내용 | 소요 시간 |
|------|------|-----------|
| 1. 접수 | 전화/메일 접수, 제품/증상 기록 | 즉시 |
| 2. 입고 | 제품 수령, 외관/구성품 검수 | 당일 |
| 3. 진단 | PC 연결 → 소프트웨어 진단 → 증상 재현 | 1~2영업일 |
| 4. 수리/교체 | 부품 수리/교체 또는 본사 RMA | 3~14영업일 |
| 5. 검수 | 캘리브레이션 + 게임 실주행 테스트 | 당일 |
| 6. 출고 | 고객 발송, 택배 번호 안내 | 당일 |

> 해외 본사 RMA 필요 시 2~4주 소요 가능 → 고객에게 사전 고지 필수

## 3.3 증상별 진단 (6개)

### (1) PC 인식 불량
- **점검**: USB 케이블/포트 교체 → 드라이버 재설치 → 다른 PC 테스트
- **브랜드별 도구**: MOZA Pit House / Fanatec 드라이버 / Logitech G HUB
- **판정**: 드라이버 문제 → 재설치, 하드웨어 → 수리/RMA

### (2) 캘리브레이션 이상
- **점검**: 소프트웨어에서 캘리브레이션 초기화 → 센서 테스트
- **판정**: 소프트웨어 리셋으로 해결 → 정상, 센서 불량 → 교체

### (3) 페달 불량
- **점검**: 로드셀/포텐셔미터 값 확인 → 케이블 연결 상태 → 센서 교체
- **판정**: 센서 값 불규칙 → 센서 교체, 케이블 단선 → 케이블 교체

### (4) 소음/진동 이상
- **점검**: 정상 FFB(Force Feedback) 소음 vs 비정상 구분
- **판정**: 모터 베어링 → 교체, 벨트/기어 마모 → 교체
- **주의**: 다이렉트 드라이브(DD) 모터 특유의 소음은 정상

### (5) 게임 호환성
- **점검**: 게임별 설정 확인 → 소프트웨어 프로파일 점검
- **판정**: 설정 문제 → 가이드 안내 (수리 대상 아님), 펌웨어 버그 → 업데이트
- **주의**: 게임별 호환성은 AS 대상이 아닌 경우 많음 → 기술지원으로 분류

### (6) 교환/반품
- **기준**: Keychron과 동일 (7일 이내 제조 결함 → 교환)
- **해외 브랜드 특이사항**: 본사 정책에 따라 교환 범위 상이 → 브랜드별 정책 확인 필수

## 3.4 브랜드별 소프트웨어 도구

| 브랜드 | 소프트웨어 | 용도 |
|--------|-----------|------|
| MOZA | Pit House | 캘리브레이션, FFB 설정, 펌웨어 업데이트 |
| Fanatec | Fanatec Control Panel | 캘리브레이션, FFB 설정, 펌웨어 |
| Logitech G | G HUB | 디바이스 관리, 프로파일, 펌웨어 |
| Playseat | - | 소프트웨어 없음 (하드웨어 전용) |

## 3.5 출고 체크리스트

- [ ] PC 인식 정상 확인 (USB 연결)
- [ ] 캘리브레이션 정상 확인
- [ ] FFB(Force Feedback) 동작 확인
- [ ] 페달 전 축 입력 확인
- [ ] 버튼/패들시프터 전체 입력 확인
- [ ] 펌웨어 최신 버전 확인
- [ ] 외관 청소 및 스크래치 확인
- [ ] 구성품 동봉 확인 (전원 어댑터, 케이블 등)
- [ ] 수리 내역서 동봉
