# PI 업데이트 — 브랜드별 발주/입고 현황 종합

[출처](https://www.notion.so/33bc0e218a4f815c8dacf098c2846870)

7개 브랜드의 Proforma Invoice(PI) 수령 및 입고 상태를 추적하는 운영 문서.

## 작성 원칙

- 모든 이슈는 PI 번호에 귀속 — 미수령 건은 `PI미수령` 표기
- 이슈 유형: 단가 / 발주수량 / 납기 / 출하 / 기타
- 상태값 4단계: ✅ 완료 / 🔄 진행중 / ⏳ 대기 / ❌ 이슈
