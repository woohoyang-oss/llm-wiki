# CS 지식베이스 종합 요약

## 개요

CS 지식베이스는 [[keychron]], [[gtgear]], Aiper, PawSwing 4개 브랜드의 고객 응대 지식을 체계적으로 정리한 문서 모음이다. 총 20개 파일로 구성되며, FAQ, AS 가이드, 반품/배송 정책 등을 포함한다. AI 기반 CS 자동 응답 및 상담원 지원 시스템의 핵심 데이터소스로 활용된다.

## Keychron 지식베이스 (2개 파일)

### faq.md

- 키크론 키보드 관련 자주 묻는 질문 모음
- 주요 카테고리: 연결/페어링, 키 매핑, 펌웨어 업데이트, 호환성
- Bluetooth 연결 문제 해결 가이드
- VIA/QMK 키 리매핑 안내
- macOS / Windows / Linux 호환성 정보

### warranty_policy.md

- 키크론 제품 보증 정책
- 보증 기간: 구매일로부터 1년
- 보증 범위: 제조 결함, 하드웨어 불량
- 보증 제외: 사용자 과실, 수리 흔적, 자연 마모
- AS 접수 절차 및 필요 서류

## GTGear 지식베이스 (12개 파일)

### 업무 가이드 (00_INDEX ~ 08_WMS)

GTGear 내부 업무 프로세스를 9개 챕터로 정리한 종합 매뉴얼:

- **00_INDEX**: 전체 목차 및 문서 구조 안내
- **01_product**: 상품 등록/수정, 옵션 관리, 가격 정책
- **02_order**: 주문 처리 플로우, 결제 확인, 주문 상태 관리
- **03_stock**: 재고 관리, 입출고 처리, 재고 실사
- **04_export_sales**: 해외 판매, 수출 프로세스, 통관 관련
- **05_crm**: 고객 관리, VIP 등급, 포인트 정책
- **06_naver**: [[naver-smartstore]] 운영 가이드, API 연동
- **07_finance_event**: 정산, 이벤트/프로모션 관리
- **08_wms_parts_misc**: WMS 연동, 부품 관리, 기타 업무

### CS 전문 가이드

- **cs_as_guide**: AS 접수/처리/완료 프로세스, 수리비 기준, 택배 수거 안내
- **cs_faq**: GTGear 고객 문의 빈출 질문, 답변 템플릿
- **cs_return_policy**: 반품/교환 정책, 반품 사유별 처리 기준, 환불 절차
- **cs_shipping_guide**: 배송 정책, 택배사별 안내, 도서산간 추가비, 출고 일정
