# Aiper (에이퍼)

## 개요

Aiper(에이퍼)는 [[gtgear]]가 국내 유통을 담당하는 로봇 풀 클리너 브랜드다. 무선 자동 수영장 청소 로봇을 제조하며, CES 2026에서 수상한 이력이 있다.

## 제품 라인업

### Scuba 시리즈

| 모델 | 특징 |
|------|------|
| Scuba SE | 엔트리 모델, 소형 풀 대상 |
| Scuba E1 | 중급형, 벽면 청소 지원 |
| Scuba S1 | 고급형, 스마트 경로 탐색 |
| Scuba X1 | 플래그십, AI 매핑 + 앱 제어 |
| Scuba V3 | 최신 모델, 개선된 흡입력 |

### 공통 기능
- 무선 충전 (코드리스)
- 자동 경로 탐색
- 필터 자동 세척 (일부 모델)
- 스마트폰 앱 연동 (상위 모델)

## 수상 이력

- **CES 2026** — Innovation Award 수상
  - 로봇 풀 클리너 카테고리

## 시장 포지셔닝

- 북미 시장이 주력, 한국 시장은 [[gtgear]]를 통해 진출
- 가정용 수영장 및 인플레이터블 풀 대상
- 가격대: 엔트리(SE)부터 프리미엄(X1)까지 다양

## 판매 채널 (국내)

- 네이버 스마트스토어
- 쿠팡
- 자사 웹사이트
