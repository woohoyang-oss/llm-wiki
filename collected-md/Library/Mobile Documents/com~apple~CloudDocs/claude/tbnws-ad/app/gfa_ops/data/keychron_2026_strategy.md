
## 5. OPS AI 자동화 로드맵

### Phase 1: 현재 (DRY_RUN)
- 매일 분석 실행 → 제안만 생성
- KB 룰 기반 점수 산출
- 수동으로 예산 조정

### Phase 2: Q2 목표 (SHADOW)
- 1건/일 자동 실행 (나머지는 제안)
- 실험 프레임워크 작동
- 자동 롤백 조건 활성화

### Phase 3: Q3~ 목표 (LIVE)
- 전체 adSet 자동 조정
- 소재 피로도 자동 감지
- 시즌/요일 계수 자동 적용
