# 키크론(Keychron) A/S 보증 정책 지침서

## 1. 무상 보증 기간

- **정품등록 완료 제품**: 구매일로부터 **2년간** 무상 A/S
- **정품등록 미완료 제품**: 구매일로부터 **1년간** 무상 A/S
- 보증기간은 `erp_serial_registration` 테이블의 `purchased_at` 기준으로 판정

## 2. 정품등록 확인 방법

MCP 도구 `get_b2c_crm_by_phone`을 호출하면 `warranty_status` 필드에서 자동 판정:
- `무상보증기간`: 구매 후 2년 이내 → **무상 수리/교환 가능**
- `보증만료`: 구매 후 2년 초과 → **유상 수리** (비용 안내)
- `구매일미확인`: 구매일 정보 없음 → **구매증빙 요청** 후 판단

### 판정 기준 데이터
- `serial_registrations`: 자동 정품등록 내역 (시리얼 번호 + 제품 정보)
- `serial_registrations_manual`: 수동 정품등록 내역 (관리자 승인 필요)
- `warranty_remaining_days`: 보증 잔여일수 (음수면 만료)

## 3. A/S 상담 시 활용 순서

### Step 1: 고객 확인
고객의 전화번호로 `get_b2c_crm_by_phone` 호출하여:
1. `customer_info` → 기본정보 (이름/이메일/주소)
2. `serial_registrations` → 정품등록 여부
3. `warranty_status` → 보증기간 판정

### Step 2: 보증기간 판정
- **무상보증기간**인 경우:
  - "고객님의 [제품명]은 [구매일] 구매로 정품등록되어 있으며, 무상 보증기간 내입니다 (잔여 N일). 무상으로 A/S 처리해드리겠습니다."
- **보증만료**인 경우:
  - "고객님의 [제품명]은 [구매일] 구매로 보증기간이 만료되었습니다. 유상 수리로 진행 가능하며, 수리 비용은 증상에 따라 안내드리겠습니다."
- **정품등록 없음**인 경우:
  - "정품등록 내역이 확인되지 않습니다. 구매증빙(영수증, 주문내역 등)을 확인해주시면 보증 여부를 판단해드리겠습니다."

### Step 3: 이전 A/S 이력 확인
- `as_records` → 기존 A/S 접수 이력
- `consultation_memos` → 이전 상담 메모
- 반복 증상이면 교환 검토, 최초 증상이면 수리 우선

### Step 4: 구매 채널 확인
- `orders` → 전채널 구매내역 (스마트스토어, 쿠팡, 지마켓 등)
- `export_logs` → ERP 출고내역 (택배사, 송장번호)
- 구매 채널에 따라 반품/교환 절차가 다를 수 있음

## 4. 무상 A/S 대상 범위

### 무상 수리 대상
- 제품 자체 결함 (키 인식 불량, 연결 끊김, LED 고장 등)
- 초기 불량 (구매 후 14일 이내)
- 정상 사용 중 발생한 하자

### 유상 수리 대상 (보증기간 내라도)
- 고객 과실에 의한 파손 (낙하, 침수, 개조 등)
- 소모품 교체 (키캡, 스위치 등 소모성 부품)
- 보증기간 만료 후 발생한 모든 증상

## 5. 특수 케이스

### 프리런칭 구매자
- `prelaunching_history`에 기록된 고객은 얼리어답터/VIP 가능성
- 프리런칭 참여 이력이 있으면 우선 응대 고려

### 복수 제품 등록
- 한 고객이 여러 제품을 등록한 경우, 각 제품별로 보증기간 독립 판정
- `warranty_status`는 제품별로 반환됨

### 시리얼 번호 없는 경우
- `serial_registrations_manual` (수동등록) 테이블에서 확인
- 수동등록은 `is_approved` = 'Y'인 경우만 유효
