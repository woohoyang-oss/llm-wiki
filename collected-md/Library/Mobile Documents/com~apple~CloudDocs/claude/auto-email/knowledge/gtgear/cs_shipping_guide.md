# 지티기어 배송 안내 가이드

## 일반 배송
- **출고 소요**: 결제 완료 후 1~2 영업일 이내
- **배송 소요**: 출고 후 1~2 영업일 (CJ대한통운)
- **총 소요 기간**: 결제 후 약 2~4 영업일
- 도서산간 지역은 추가 1~2일 소요

## 배송비
- 기본 배송비: 무료 (일정 금액 이상 구매 시) 또는 3,000원
- 도서산간 추가 배송비: 별도 안내
- 해외 배송: 불가 (국내 배송만)

## 배송 추적
- 출고 시 네이버 스마트스토어에 운송장 자동 등록
- 운송장 번호로 CJ대한통운 홈페이지에서 추적 가능
- "배송 중" 상태에서 1~2일 내 수령 예정

## 배송 지연 안내
- 주문 폭주, 명절 등 특수 기간에는 출고가 지연될 수 있음
- 재고 부족 시 입고 예정일 안내 후 발송
- 천재지변, 택배사 사정으로 인한 지연은 택배사 문의 안내

## 수령 후 확인 사항
- 수령 즉시 제품 상태 확인 (외관 파손, 구성품 누락 등)
- 이상 발견 시 수령 후 7일 이내 고객센터로 연락
- 배송 중 파손이 의심되는 경우 포장 상태 사진 촬영 권장

## 출고 전 주문 변경
- 출고 전: 배송지 변경, 수량 변경 가능
- 출고 후: 변경 불가 (반품 후 재주문 안내)

## 자주 묻는 배송 질문

### "주문했는데 아직 출고가 안 됐어요"
→ 결제 완료 후 1~2 영업일 이내 출고됩니다. 주말/공휴일은 출고일에서 제외됩니다.

### "운송장 번호가 안 나와요"
→ 출고 처리 시 자동으로 운송장이 등록됩니다. 출고 전이라면 조금만 기다려주세요.

### "배송 중인데 언제 오나요?"
→ CJ대한통운 기준 보통 1~2 영업일 내 배달됩니다. 운송장 번호로 실시간 추적 가능합니다.

### "부분 배송이 가능한가요?"
→ 일부 상품이 재고 부족인 경우, 먼저 발송 가능한 상품을 선배송하고 나머지는 입고 후 발송할 수 있습니다. 개별 안내 드립니다.
