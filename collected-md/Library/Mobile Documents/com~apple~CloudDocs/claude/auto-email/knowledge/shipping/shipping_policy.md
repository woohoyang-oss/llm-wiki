# 배송 정책 통합 가이드 (전 브랜드 공통)

## 배송 채널
- **CJ대한통운**: 키크론, 지티기어 기본 배송사
- **로젠택배**: 일부 상품 (사방넷 경유)
- **직접배송**: 대형 상품 (레이싱 휠 거치대 등)

## 배송 소요 시간
| 단계 | 소요 시간 |
|------|----------|
| 결제 → 출고 | 1~2 영업일 |
| 출고 → 배송 완료 | 1~2 영업일 |
| **합계** | **2~4 영업일** |

- 도서산간 지역: 추가 1~2일
- 주말/공휴일: 출고 불가 (익영업일 출고)

## 주문 상태 값
| 상태 | 설명 |
|------|------|
| 주문접수 | 결제 완료, 아직 출고 전 |
| 배송중 | 운송장 발급 후 배송사 인수 |
| 배송완료 | 수령 확인 |
| 취소 | 취소 처리됨 |

## 자주 묻는 배송 문의 유형

### 1. "배송 언제 오나요?"
- 결제 후 1~2 영업일 출고, 출고 후 1~2일 배송
- 주말/공휴일 출고 불가 안내
- 운송장 등록 후 네이버 스마트스토어에서 배송추적 가능

### 2. "운송장 번호 알려주세요"
- 네이버 스마트스토어 → 주문내역에서 확인 가능
- 운송장 미발행 시: 출고 전 상태, 1~2 영업일 내 출고 예정 안내

### 3. "배송지 변경하고 싶어요"
- 출고 전: 변경 가능 → 주문번호 + 변경 배송지 안내 요청
- 출고 후: 배송사 직접 연락 또는 반품 후 재주문

### 4. "배송이 지연되고 있어요"
- 택배사 물량 폭주, 기상 악화 등 외부 요인 가능
- 배송 추적 링크 안내
- 3일 이상 지연 시 택배사 확인 후 고객 연락

### 5. "택배사 변경 가능한가요?"
- 기본 CJ대한통운 사용
- 택배사 변경은 불가능한 경우가 대부분
- 특수 사정 시 상담 후 개별 대응

### 6. "부분 배송되었어요"
- 주문 상품이 다른 물류센터에서 출발 시 분할 배송 가능
- 나머지 상품도 순차 발송 안내

## 배송비 정책
| 구분 | 금액 |
|------|------|
| 기본 배송비 | 무료 (일정 금액 이상) 또는 3,000원 |
| 도서산간 추가 | 3,000~5,000원 |
| 교환/반품 배송비 (단순변심) | 왕복 6,000원 고객 부담 |
| 교환/반품 배송비 (불량/오배송) | 무료 |

## 응대 템플릿

### 배송 일정 안내
"주문해주신 상품은 결제 확인 후 1~2 영업일 내에 출고되며, 출고 후 1~2 영업일 이내 수령 가능합니다. 주말 및 공휴일에는 출고가 진행되지 않는 점 참고 부탁드립니다."

### 배송 지연 안내
"현재 택배사 사정으로 배송이 다소 지연될 수 있는 점 양해 부탁드립니다. 운송장 번호로 실시간 배송 현황을 확인하실 수 있습니다."

### 배송 추적 안내
"네이버 스마트스토어 → 주문내역에서 운송장 번호와 배송 상태를 확인하실 수 있습니다."

## 배송 관련 데이터 조회 방법 (내부용)
- 주문 현황: `get_sales_overview` 또는 `get_sales_daily` 활용
- 클레임 현황: `get_claims_summary` 활용 (취소/반품/교환)
- 미답변 문의: `get_cs_overview` 또는 `get_qna_unanswered` 활용
- 재고 확인: `get_inventory_current` 활용
