# 에이퍼(Aiper) AS 처리 가이드

## AS 접수 채널
- **이메일**: support@gtgear.co.kr
- **네이버 스마트스토어**: 1:1 문의

## AS 접수 시 필요 정보
- 주문번호
- 제품 모델명 (예: Scuba S1, Scuba V3 Pro 등)
- 구매일자
- 증상 설명 + 사진/영상 (가능하면)
- 연락처

## 보증 정책
| 구분 | 기간 | 비용 |
|------|------|------|
| 무상 수리 | 구매일로부터 1~2년 (모델별 상이) | 무료 |
| 유상 수리 | 보증기간 경과 | 부품비 + 수리비 |
| 사용자 과실 | 보증기간 무관 | 유상 수리 |

## 보증 제외 사항
- 떨어뜨림, 극단적 온도, 부적절한 사용으로 인한 손상
- 비공식 판매처에서 구매한 제품
- 소모품 (필터, 브러시 등)

## AS 처리 절차
1. **접수**: 고객 문의 확인 + 증상 파악
2. **1차 트러블슈팅**: 전화/메일로 자가 해결 안내
3. **제품 회수**: 해결 불가 시 제품 수거 (택배)
4. **진단**: 기술팀 진단 (무상/유상 판별)
5. **수리/교환**: 진단 결과에 따라 수리 또는 교환
6. **발송**: 수리 완료 후 고객에게 재발송

## 주요 부품 / 소모품
- 필터 카트리지/필터백 — 정기 교체 권장 (3~6개월)
- 브러시 — 마모 시 교체
- 배터리 — 수명 저하 시 교체 (유상)
- 충전 독 — 고장 시 교체

## 자주 발생하는 AS 사유
1. 배터리 수명 저하 / 충전 불가
2. 모터 이상 (소음, 작동 불량)
3. 흡입력 저하
4. 방수 불량 / 누수
5. 소프트웨어 오류 (AI 모델)

## 고객 응대 템플릿

### 접수 시
"AS 접수 감사합니다. 정확한 진단을 위해 주문번호, 제품 모델명, 그리고 증상을 사진/영상과 함께 보내주시면 빠르게 도와드리겠습니다."

### 트러블슈팅 안내 시
"말씀하신 증상에 대해 먼저 간단한 점검을 안내드립니다. [점검 내용]. 시도 후에도 동일 증상이면 제품 수거 후 정밀 진단을 진행하겠습니다."

### 수리 완료 시
"수리가 완료되어 오늘 발송해 드렸습니다. 운송장 번호는 [번호]이며, 1~2 영업일 내 수령 가능합니다. 추가 문의사항이 있으시면 언제든 연락주세요."
