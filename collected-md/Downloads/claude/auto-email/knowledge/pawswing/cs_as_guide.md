# 포스윙(PawSwing) AS 처리 가이드

## AS 접수 채널
- **이메일**: support@gtgear.co.kr
- **네이버 스마트스토어**: 1:1 문의

## 보증 정책
| 제품 | 보증기간 | 비고 |
|------|---------|------|
| PawSwing Neo | 2년 보증 + 90일 사용 보장 | 양도 불가 (원구매자 한정) |
| AutoComb Pro | 평생 보증 (Lifetime Warranty) | 양도 불가 |
| 소모품 (빗, 펠트, 건조제) | 보증 제외 | 별도 구매 |

## 보증 범위
### 보증 대상
- 제조 결함이 있는 부품
- 정상 사용 중 발생한 고장
- 수리 또는 교체 무상 제공

### 보증 제외
- 고양이에 의한 손상 (물어뜯음, 긁힘, 찢어짐, 움푹 파임)
- 사용자 과실, 남용, 부주의
- 무단 수리 또는 개조
- 자연재해
- 일반적인 마모
- 비공식 판매처 구매 제품
- 개인 부상, 재산 피해, 반려동물 피해

## AS 처리 절차
1. **접수**: support@gtgear.co.kr로 문의
2. **증상 설명**: 구매 주문번호 + 문제 설명 + 사진
3. **트러블슈팅**: 원격 해결 시도
4. **RMA 발급**: 해결 불가 시 반품 승인 번호(RMA) 발급
5. **제품 발송**: RMA 번호 확인 후 제품 발송
6. **수리/교환**: 약 10영업일 내 처리
7. **발송**: 수리 완료 후 재발송

## 자주 발생하는 AS 사유
1. **빗 파손/마모** — 소모품이므로 교체 안내 (유상)
2. **펠트 훼손** — 고양이 스크래치/물어뜯음 → 보증 제외
3. **센서 오작동** (Pro) — 보증 대상
4. **간식 디스펜서 고장** (Pro) — 보증 대상
5. **구조물 변형** — 원인에 따라 무상/유상 판별

## 고객 응대 시 주의사항
- 고양이에 의한 손상은 보증 대상이 아님을 정중히 안내
- Neo는 전기 제품이 아니므로 대부분 물리적 손상이 AS 사유
- Pro의 센서/디스펜서 관련 문의가 가장 많음
- 소모품 교체 주기(3~6개월)를 사전에 안내하여 클레임 예방

## 고객 응대 템플릿

### 접수 시
"문의 감사합니다. 정확한 확인을 위해 주문번호와 제품 상태 사진을 보내주시면 빠르게 도와드리겠습니다."

### 고양이 손상으로 보증 불가 시
"확인 결과, 반려동물에 의한 물리적 손상으로 판단되어 보증 수리 범위에 해당하지 않습니다. 교체 부품을 별도 구매하실 수 있도록 안내드리겠습니다."

### 보증 수리 진행 시
"제조 결함으로 확인되어 무상 수리/교환 처리해 드리겠습니다. RMA 번호를 발급해 드렸으니, 해당 번호를 기재하여 제품을 보내주세요."
