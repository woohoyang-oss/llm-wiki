# ⚔️ 아제로스 MUD — WoW 클래식 텍스트 어드벤처 게임 기획서

> 월드 오브 워크래프트 클래식의 세계관을 텍스트 MUD로 재현하는 멀티플레이 브라우저 게임

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 텍스트 기반 멀티유저 던전 (MUD) |
| **테마** | WoW 클래식 — 얼라이언스 인간 시작 지역 |
| **접속 방식** | 웹 브라우저 (WebSocket 실시간 통신) |
| **기술 스택** | Node.js + Express + WebSocket + SQLite |
| **동시 접속** | 다수 유저 실시간 상호작용 |
| **시작 지역** | 노스샤이어 수도원 → 골드샤이어 → 엘윈 숲 → 스톰윈드 |

### 4.2 어그로(위협) 시스템 — "마지막 타격자 = 탱커"

핵심 메카닉: **몬스터는 자신을 마지막으로 타격한 플레이어를 집중 공격합니다.**

```
파티 전투 예시:

━━━ ⚔️ 파티 전투! ━━━
호거 (Lv.11 정예) — HP: 800/800

[1턴] 
  용사김(전사)이 호거를 공격합니다! → 18 데미지
  → 호거의 시선이 용사김에게 집중됩니다! ← 어그로
  힐러영희(사제)가 [갱신]을 용사김에게 시전합니다!
  도적철수(도적)가 호거를 공격합니다! → 25 데미지
  → 호거의 시선이 도적철수에게 바뀝니다! ← 어그로 전환!
  
  호거가 도적철수를 공격합니다! → 45 데미지 💥

[2턴]
  용사김(전사)이 호거를 공격합니다! → 16 데미지
  → 호거의 시선이 용사김에게 돌아옵니다! ← 탱킹 성공!
  힐러영희(사제)가 [하급 치유]를 도적철수에게 시전합니다! → +30 HP
  
  호거가 용사김을 공격합니다! → 38 데미지
```

**어그로 규칙:**
- 몬스터는 **가장 마지막에 자신을 때린 플레이어**를 공격합니다
- 전사의 [도발] 스킬은 강제로 어그로를 가져옵니다
- 사제의 [소멸]은 자신에게 온 어그로를 제거합니다
- 파티원들은 전사가 먼저 때린 후 딜을 시작하는 게 이상적입니다
- 보스급 몬스터는 어그로 전환 시 전환 대상에게 추가 데미지를 줍니다
