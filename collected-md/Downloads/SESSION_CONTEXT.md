# Keychron SmartStore 분석 세션 컨텍스트

> 이 파일을 새 세션에 붙여넣으면 이전 작업을 이어서 진행할 수 있습니다.

## 프로젝트 개요
키크론(Keychron) 네이버 스마트스토어의 매출, 마케팅, 광고 데이터를 종합 분석하여 HTML 대시보드를 만들고 있습니다.

## 분석 기간
- 전체: 2024.03 ~ 2026.02 (약 24개월)
- 핵심 비교: **Period A (2024.11~2025.02)** vs **Period B (2025.11~2026.02)**

## 3-Phase 광고 프레임워크
- **Phase 1 (2024.03~2025.02)**: SA 중심, 기존 유저 타겟, r=-0.937 (역상관)
- **Phase 2 (2025.03~2025.10)**: 효율화 축소기, r=0.454
- **Phase 3 (2025.11~2026.02)**: GFA 확대, 신규타겟(여성 20-35 사무직), r=0.872

### 핵심 발견
1. **슈퍼적립 소멸**: 네이버가 밀어줬던 무료 트래픽(~1.3억) 사라짐 → 키크론 문제 아닌 네이버 정책 변화
2. **플랫폼 전환**: 네이버쇼핑-광고(28.4%→0%) → 플러스스토어(4.5%→17.2%) + ADVoost(0→7.6%)
3. **GFA 최적 예산**: 월 1,500~2,000만. 초과 시 클릭당 매출 급락 (26/01 3,666만 투자 → 클릭당 매출 3,144원)
4. **전환 병목**: 클릭 +77% vs 주문 -13%. 장바구니→구매 전환율 38.9%→28.3%로 하락

## 기술적 주의사항
- **HTML 생성 시 f-string 금지**: Python f-string과 JS template literal(`${}`) 충돌 → 빈 HTML 렌더링. 반드시 string concatenation 사용.
- **scipy 미설치**: `numpy.corrcoef()` 사용
- **GFA 컬럼명**: '총 전환 매출액' (띄어쓰기 주의)
- **pip**: `--break-system-packages` 플래그 필요
