# 키워드/트렌드 분석

## 1단계: 데이터 수집 (병렬 실행)

다음 MCP 도구를 병렬로 호출하세요:

- `get_top_keywords` — 상위 키워드
- `get_trend_analysis` — 트렌드 분석
- `get_revenue_ad_correlation` — 매출-광고 상관관계
- `get_sales_daily` — 일별 매출 추이
- `get_sa_performance` — 검색광고 성과

## 2단계: 분석 및 리포트

### 리포트 형식

```
📈 트렌드 분석 리포트 (YYYY-MM-DD)

■ 상위 키워드 TOP 10
| 순위 | 키워드 | 검색량 | 전환율 | 추이 |
|------|--------|--------|--------|------|
| 1    | ...    | ...    | ...    | ↑/↓  |

■ 트렌드 변화
- 급상승 키워드: [목록]
- 하락 키워드: [목록]
- 신규 진입 키워드: [목록]

■ 매출-광고 상관관계
- 광고비 증가 → 매출 증가 상품: [목록]
- 광고비 대비 매출 효율 낮은 상품: [목록]
- 자연 유입 매출 비중: X%

■ 인사이트
1. [핵심 발견사항]
2. [기회 요인]
3. [리스크 요인]

■ 액션 제안
1. [키워드 전략 제안]
2. [광고 예산 조정 제안]
3. [상품 기획 제안]
```

## 3단계: 심화

- 특정 상품/카테고리 트렌드 요청 시 해당 데이터만 필터링
- 경쟁사 키워드 비교 요청 시 관련 데이터 추가 분석
- 시즌성 분석이 필요하면 과거 데이터 추이 비교
