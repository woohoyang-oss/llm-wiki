# 상품 원가/마진 분석

## 1단계: 데이터 수집 (병렬 실행)

다음 MCP 도구를 병렬로 호출하세요:

- `get_margin_analysis` — 마진 분석
- `get_problem_products` — 문제 상품
- `get_product_cost` — 상품 원가 (특정 상품 조회 시)
- `get_products_cost_batch` — 상품 원가 일괄 조회
- `get_product_real_price` — 실판매가
- `get_sales_by_product` — 상품별 매출

## 2단계: 분석 및 리포트

### 리포트 형식

```
💰 마진 분석 리포트 (YYYY-MM-DD)

■ 전체 요약
- 평균 마진율: X%
- 마진율 목표 대비: +/-X%p
- 문제 상품 수: X개

■ 상품별 마진 분석
| 상품명 | 원가 | 실판가 | 마진율 | 월매출 | 월이익 | 상태 |
|--------|------|--------|--------|--------|--------|------|
| ...    | ...  | ...    | ...    | ...    | ...    | ...  |

■ 문제 상품 🚨
[마진율이 낮거나 적자인 상품]
| 상품명 | 마진율 | 문제 원인 | 개선 제안 |
|--------|--------|----------|----------|
| ...    | ...    | ...      | ...      |

■ 수익성 개선 제안
1. 가격 조정 필요 상품: [목록]
2. 원가 절감 검토 상품: [목록]
3. 판매 중단 검토 상품: [목록]

■ 채널별 마진 비교
| 채널 | 평균 마진율 | 수수료 | 순마진 |
|------|-----------|--------|--------|
| ...  | ...       | ...    | ...    |
```

## 3단계: 심화 분석

- 사용자가 특정 상품을 지정하면 `get_product_cost`, `get_product_real_price`로 상세 조회
- 비즈머니 현황이 필요하면 `get_bizmoney` 추가 호출
- 모델별 분석 요청 시 `get_sales_by_model` 활용
