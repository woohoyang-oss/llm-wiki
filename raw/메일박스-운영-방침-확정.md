# 메일박스 운영 방침 (2026-03-23 확정)

## 방식: 현행 유지 (포워딩 + Send As)

## 새 브랜드 메일함 추가 절차
1. mail.tbe.kr → 메일박스 → "새 메일박스 추가" → brand/category/address 입력
2. 새 브랜드 Gmail에서: 설정 → 전달 → ai@tbe.kr로 포워딩 설정
3. ai@tbe.kr Gmail에서: 설정 → "다른 주소에서 메일 보내기" (Send As) → 새 브랜드 주소 추가
4. 완료 — MailboxResolver가 X-Forwarded-To로 자동 매칭

## 코드 변경 불필요
- MailboxResolver: DB에서 동적 조회 (하드코딩 없음)
- GmailService: Send As로 브랜드 주소 발송
- pipeline.py: MailboxResolver → AI → 발송 자동 처리
