## 2. 시드 매칭도 평가

### 잘 커버되는 영역
- **Keychron A/S**: 지식문서(FAQ, AS발송절차), 템플릿(AS접수, 수리완료, 시리얼확인), 자동화(AS폼감지), 배정규칙(키크론AS→신희웅) — **완벽 커버**
- **GTGear A/S**: 지식문서(GTGear AS 가이드), 자동화(GTGear 폼감지), 배정규칙(지티기어CS→강은비) — **커버됨**
- **교환/반품**: 지식문서(교환반품정책), 템플릿(교환반품답변, 반품접수), 매크로(교환반품안내) — **커버됨**
- **스팸 필터링**: 자동화(스팸필터), 태그(스팸) — **커버됨**

### 부족한 영역
1. **채널 파트너 CS (현대이지웰, 쿠팡 등)**: 상품문의 답변, 상품 확인 요청 대응 → **템플릿/지식문서 없음**
2. **세금계산서 발행 요청**: B2B 세금계산서 처리 절차 → **지식문서 없음** (B2B 견적 가이드는 있으나 세금계산서 발행 절차 없음)
3. **배송의뢰서 (G마켓 등)**: 마켓 발주/배송의뢰 처리 → **자동 분류/태깅 없음**
4. **시스템 알림 자동 무시**: 인증번호, OTP, 정산알림 등 자동으로 무시/아카이브 처리하는 자동화 규칙 없음
