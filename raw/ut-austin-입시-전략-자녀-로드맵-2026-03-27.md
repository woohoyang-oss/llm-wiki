## 2. 목표 대학

### University of Texas at Austin (UT Austin)
- 타겟 전공: **B.S. Arts and Entertainment Technologies (AET)**
- 소속: College of Fine Arts → School of Design and Creative Technologies (SDCT)
- 위치: Austin, Texas, USA

## 3. AET 전공 개요 (공식 확인 완료 ✅)

### 전공 정의
> "기술적 마인드셋으로 디지털·물리적 공간에서 의미 있는 경험을 창조하는 것"

### 전공 트랙
| 트랙 | 내용 |
|------|------|
| Digital Visualization | 3D 모델링, 캐릭터 애니메이션, 모션 그래픽, 컨셉아트 |
| Game Design | 게임 경험 설계, 극한 제약 속 플레이 경험 창조 |
| Music & Sound | 오디오 프로세싱, 사운드 디자인 |
| PLAI (Projection, Lighting & Interactivity) | 라이브 이벤트, 몰입형 미디어, 인터랙티브 시스템 |

### 사용 툴
- Adobe Creative Cloud
- Autodesk Maya (블렌더 경험 → 자연스럽게 연결)
- Unity
- 기타 업계 표준 소프트웨어

### 졸업 후 커리어
크리에이티브 테크놀로지스트, 게임 디자이너, 3D 아티스트, 사운드 디자이너, 경험 디자이너, 콘텐츠 크리에이터, 라이팅 디자이너 등

## 4. AET 입시 요건 (공식 확인 완료 ✅)

| 항목 | 내용 |
|------|------|
| 포트폴리오 | **요구하지 않음** (제출도 불가) |
| 핵심 | UT Austin 지원서 에세이 + 활동 기술서 |
| 에세이 방향 | AET 관련 관심사, 창의적 참여 경험, 수강 이력 구체적으로 기술 |
| Creative Prompt | 내부 편입 시 필수 (신입 지원 시는 에세이로 대체) |
| 지원 시기 | Fall 학기만 (Regular Deadline, Early Action 제외) |
| GPA | 3.0 이상 권고 |
| SAT | Test-optional이나 제출 시 유리 (목표: 1350~1400) |
| TOEFL | 79점 이상 (목표: 100+) |

> **핵심 전략:** 로블록스·마인크래프트·블렌더 경험을 에세이 소재로 구체적으로 작성하는 것이 합격의 열쇠

## 6. 고교 경로 추천

### 결론: 성남외고보다 일반고 + 포트폴리오 전략 권장

| 경로 | 평가 |
|------|------|
| 성남외고 | 영어 환경은 좋으나, AET 준비에 비효율적. 포트폴리오 쌓기 어려움 |
| 과학고/자사고 | 공부 동기 고려 시 현실적으로 어려움 |
| **일반고 + 독학 포트폴리오** | ✅ 추천 — 내신 관리하면서 블렌더·코딩·AI 포트폴리오 자유롭게 구축 |
| 국제학교 | 최적 환경이나 비용 부담 큼 |

> AET는 포트폴리오를 공식 요구하지 않지만, 에세이에 녹여낼 **실제 경험과 프로젝트**가 많을수록 유리
